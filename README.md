# algoritimos-e-estrutura-de-dados

Este repositório reúne o material prático e as implementações desenvolvidas durante a disciplina algoritimos e estruturas de dados. O objetivo é consolidar conceitos fundamentais de organização de dados e eficiência de algoritmos utilizando a linguagem Java.

🗂️ Conteúdo e Atividades Implementadas

Abaixo está o roteiro de todas as estruturas e algoritmos desenvolvidos neste repositório:

🧱 Estruturas de Dados Lineares

Listas Simplesmente Encadeadas

Refatoração de Listas utilizando Tipos Genéricos

Pilhas (Stacks)

Filas (Queues)

🌳 Estruturas de Dados Não-Lineares e Hash

Árvores Binárias (Binary Trees)

Árvores Binárias de Busca (BST - Binary Search Trees)

Tabelas Hash (Hash Tables)

⚙️ Aplicações Práticas e Algoritmos Específicos

Algoritmos para Transformação de Expressões Aritméticas (Aplicações práticas com Pilha)

Simulação de Fila de Supermercado com controle de tempo de chegada (Aplicações práticas com Fila)
