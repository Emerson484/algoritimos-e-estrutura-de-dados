# algoritimos-e-estrutura-de-dados
Repositório contendo as atividades práticas e implementações de estruturas de dados desenvolvidas na disciplina de Algoritmos e Estruturas de Dados (2025). Inclui implementações em Java de Listas, Pilhas, Filas, Árvores (Binária, Busca, AVL, N-ária) e Tabelas Hash.
